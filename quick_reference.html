Rujukan Pantas Arahan Perintah (Cheat Sheet)

Salinan 1-Klik

Satu halaman ringkas yang menggabungkan semua sintaks harian penting. Pastikan anda melihat label persekitaran (PowerShell / WSL Ubuntu) sebelum menyalin.

Kawalan Hos Windows & Sesi WSL

Semak status pemasangan & versi WSL (Windows PowerShell) wsl --status
Log masuk ke distro lalai (Windows PowerShell) wsl -d Ubuntu
Matikan semua sesi Linux untuk bersihkan RAM (Windows PowerShell) wsl --shutdown

Pengendalian Daemon & Servis Docker

Aktifkan servis latar belakang Docker Engine (WSL Ubuntu) sudo service docker start
Semak status kesihatan semasa servis Docker (WSL Ubuntu) sudo service docker status

Pengurusan Harian Kontena (Container Operations)

Luncurkan pelayan Nginx baharu di latar belakang (Port 8080) docker run -d -p 8080:80 --name web-node nginx
Lihat senarai kontena yang sedang AKTIF sahaja docker ps
Lihat SEMUA senarai kontena (Aktif & Berhenti) docker ps -a

Pembersihan Storan & Sisa Sistem

Hentikan kontena secara selamat docker stop web-node
Padam kontena secara paksa (Force Delete) docker rm -f web-node
Nuke/Padam total semua cache imej, volume, & rangkaian tidak aktif (Bebaskan NVMe Disk) docker system prune -a --volumes -f
