00. Proses Mula & Penyediaan Persekitaran

Langkah awal yang perlu dilakukan pada sistem Windows sebelum mengaktifkan atau menguruskan WSL2 dan kontena Docker.

1. Buka PowerShell 7 sebagai Administrator

Klik kanan pada ikon PowerShell 7 dan pilih Run as Administrator untuk kebenaran penuh.

2. Semak Status Pemasangan WSL Sedia Ada wsl --status
3. Lancarkan Terminal Ubuntu (Gunakan Default User) wsl -d Ubuntu
